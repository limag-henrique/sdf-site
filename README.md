Bem-vindo à apresentação do redesign do site da Sala de Demonstrações de Física (SDF) da UFMG. Este projeto consiste no aprimoramento visual e estrutural de um site já existente, desenvolvido para fins de estudo.

Todos os créditos referentes às fotografias, conteúdos e pesquisas pertencem exclusivamente ao portal gerenciado pelo Departamento de Física da UFMG. Para acessar o conteúdo original e verificar as informações, visite: https://demonstracoes.fisica.ufmg.br/
